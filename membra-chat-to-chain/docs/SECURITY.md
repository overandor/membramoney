# MEMBRA Security

## Principles
- No private keys stored
- No raw KYC on-chain
- Manual approval gates for mainnet
- Policy engine blocks secrets and PII

## Policy Checks
- Secret detection (private keys, seed phrases)
- PII detection (emails, phones)
- Blocked phrase detection (guaranteed profit, etc.)

## Mainnet Safety
- Payment payload preparation only
- Explicit human/multisig signature required
- No auto-signing
